foo_cad

CD Art Display/Rainmeter interface plugin for foobar2000.

GitHub project: poiru/foo-cad

Installation

  • Ensure that foobar2000 1.1.11 or higher is installed
  • Download foo_cad 1.0.2
  • Run (double-click) the downloaded file to install

Note: If you are running a portable version of foobar2000, use Library → Configure → Components → Install.... Then, select the downloaded foo_cad.fb2k-component file to install.
